# Web Interface (Flask)

A simple web UI is available to run the same pipeline as `app.py`.

## Run
1. Install Flask (no other deps):
```bat
pip install flask
```
2. Start the server:
```bat
python interface.py
```
3. Open in browser:
- http://127.0.0.1:5000

## Notes
- This is a demo and not medical advice.
- If you want the UI to auto-run on startup, add a small launcher script.
